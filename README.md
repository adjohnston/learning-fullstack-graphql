# Learning Fullstack Graphql

Getting an understanding of how to create a server &amp; client side using graphql.

## The Stack

### Backend

* Apollo Server
* Zeit Micro
* Docker
* MongoDB/Mongoose

### Frontend

* Apollo Client
* Create React App (there's an issue with Apollo Link which caused me to use it)

### Hosting

* Zeit Now.
